HotStreams is a radio directory module for [LMP](/plugins-lmp) and
similar plugins.\
\
It supports the following radios:

-   Digitally Imported.
-   SkyFM.
-   SomaFM.
-   Huge radio directory from [dir.xiph.org](http://dir.xiph.org).
-   42fm.
